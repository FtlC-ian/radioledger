Welcome to RadioLedger

Let's get your ham radio logging station set up in a few easy steps. This wizard will help you configure:

  • 🔐 Secure login to your RadioLedger account
  • ☁️ Download your existing logbook from the cloud
  • 📡 Detect WSJT-X, JS8Call, and N1MM+ software
  • 🔌 Configure UDP listeners for digital modes
  • 🏆 Set up LoTW automatic upload signing
  • 🎛️ Configure rig control (Flrig / Hamlib)

You can skip optional steps and configure them later in Settings.
