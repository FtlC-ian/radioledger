# Getting Started with RadioLedger

> Get from zero to your first logged QSO in under 5 minutes.

RadioLedger is an open-source ham radio logging platform you run on your own
infrastructure. This guide starts with [self-hosting with Docker](../self-hosting/index.md).

## Prerequisites

- A ham radio license (any class)
- Your callsign
- Docker installed for self-hosting

## Get Running

### 🖥️ Self-Hosted

Run RadioLedger on your own hardware. Requires Docker Compose.

→ **[Installation guide](installation.md)**

### 📥 Already Have Logs?

Import your existing log from any ADIF-compatible logger.

→ **[Import your ADIF file](import-existing-log.md)**

## What's Next?

After your first QSO, connect external services:

- **[Connect LoTW, QRZ, eQSL, and more](connect-services.md)** — automate confirmation tracking
- **[Install the desktop client](../desktop/index.md)** — for WSJT-X / FT8 auto-logging
- **[Install the mobile app](../mobile/index.md)** — for POTA and SOTA activations
