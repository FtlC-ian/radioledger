# Glossary

> Ham radio terms for operators new to the hobby, and logging-specific terms used in RadioLedger.

## A

**ADIF** (Amateur Data Interchange Format)
: The standard file format for exchanging ham radio log data. Used by virtually all logging software. Files use `.adi` (text) or `.adx` (XML) extensions.

**ADIF DXCC entity number**
: A numeric ID assigned to each DXCC entity in the ADIF specification (e.g., 291 = USA, 339 = Japan).

**Award**
: A recognition given by ARRL, CQ, or other organizations for achieving specific operating milestones (e.g., working 100 DXCC entities for DXCC, all 50 states for WAS).

## B

**Band**
: A range of radio frequencies allocated to amateur radio use. Common HF bands: 80m, 40m, 20m, 15m, 10m. VHF/UHF: 6m, 2m, 70cm.

**Bureau** (QSL bureau)
: A bulk QSL card exchange system operated by national amateur radio organizations. Cards are batched and sent by country, reducing postage costs.

## C

**Cabrillo**
: The standard file format for contest log submission. Generated by RadioLedger from contest session data.

**CQ zone**
: One of 40 geographic zones defined by CQ Magazine for contest and award purposes. Each zone covers a broad geographic area.

**DXCC** (DX Century Club)
: ARRL's prestige award for working at least 100 DXCC entities. One of the most sought-after awards in amateur radio.

**DXCC entity**
: A country, territory, or geographic area recognized by ARRL for DXCC award credit. Currently approximately 340 entities.

## E

**eQSL**
: An electronic QSL card exchange service at eqsl.cc.

## F

**FT8**
: A highly popular weak-signal digital mode developed by Joe Taylor (K1JT) and Steve Franke (K9AN). Operates in 15-second protocol cycles and is widely used on HF bands.

**FT4**
: A faster variant of FT8 with 7.5-second protocol cycles, designed for contesting.

## G

**Grid square**
: A geographic location reference using the Maidenhead Locator System. A 4-character grid (e.g., `FN42`) covers a 2° longitude × 1° latitude area. See [Maidenhead Grids](maidenhead-grids.md).

## H

**Ham radio** / **Amateur radio**
: A licensed radio communication hobby regulated by the FCC (in the USA) and equivalent agencies worldwide.

## I

**IOTA** (Islands on the Air)
: An award program for working amateur radio operators on islands worldwide.

**ITU zone**
: One of 90 zones defined by the International Telecommunication Union.

## J

**JS8Call**
: A digital messaging mode based on FT8, allowing keyboard-to-keyboard text communication over HF.

## L

**LoTW** (Logbook of The World)
: ARRL's online QSL confirmation system. The gold standard for award confirmation, required for DXCC, WAS, and other ARRL awards.

**Logbook**
: A collection of QSO records. In RadioLedger, each logbook is independent with its own settings, sync configuration, and award tracking.

## M

**Maidenhead Locator** (grid square)
: See *Grid square*.

**Mode**
: The modulation type used for a radio contact. Common modes: SSB (phone), CW (Morse code), FT8 (digital), FM (repeaters).

**Multiplier**
: In contests, a category of stations that counts toward your score. Each new multiplier (e.g., each new DXCC entity, US state, or CQ zone) multiplies your total QSO points.

## O

**Operator**
: The person at the controls of a radio station. In RadioLedger, an operator is distinct from a station callsign — multiple operators can share a callsign (club station), and one operator can have multiple callsigns.

## P

**PKCE** (Proof Key for Code Exchange)
: A security extension to OAuth2 used by RadioLedger's desktop and mobile clients for secure authentication without storing client secrets.

**POTA** (Parks on the Air)
: A wildly popular program where ham operators activate parks and wildlife areas. Requires 10+ QSOs for a valid activation.

**Propagation**
: How radio waves travel between stations. HF propagation is heavily affected by the ionosphere, solar activity, time of day, and season.

## Q

**QSL** (card/confirmation)
: Confirmation that a QSO took place. Can be via paper card, LoTW, eQSL, or QRZ logbook.

**QSO**
: A completed two-way radio contact. The fundamental unit of ham radio logging.

**QTH**
: Ham radio shorthand for location/address.

**QRZ**
: QRZ.com — a callsign lookup service and ham radio social network with optional logbook sync.

**QRP**
: Low-power operation, typically 5W or less. `/QRP` suffix is sometimes used by operators to indicate QRP.

## R

**RST**
: A signal report consisting of Readability (1-5), Signal strength (1-9), and Tone (1-9 for CW only). Voice contacts typically use 59 ("five nine"); CW uses 599.

**RLS** (Row-Level Security)
: A PostgreSQL feature that RadioLedger uses to ensure each user can only access their own data — even at the database level.

## S

**SOTA** (Summits on the Air)
: An award program for activating mountain summits by radio.

**SSB** (Single-Sideband)
: The standard voice mode for HF amateur radio. Upper sideband (USB) is used on HF bands 20m and above; Lower sideband (LSB) on 40m and below.

**Station callsign**
: In RadioLedger, distinct from the operator. A callsign can be assigned to a person, a club, or a special event station.

## T

**tQSL**
: The ARRL's desktop application for signing logs for LoTW upload. Required for LoTW. The RadioLedger desktop client integrates with tQSL.

## U

**UTC** (Coordinated Universal Time)
: The universal time reference used in amateur radio. All RadioLedger timestamps are stored in UTC.

## V

**VUCC** (VHF/UHF Century Club)
: ARRL award for working 100+ grid squares on VHF bands.

## W

**WAS** (Worked All States)
: ARRL award for working all 50 US states. Confirmed via LoTW or paper QSL.

**WSJT-X**
: Software suite by K1JT and team for weak-signal digital modes including FT8, FT4, WSPR, JT65. RadioLedger integrates via WSJT-X's UDP protocol.
