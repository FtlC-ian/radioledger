Pick the path that matches how you operate.

- **New operator or new account:** start with [Getting Started](./getting-started/) and log one real QSO with UTC time, band, mode, and signal reports.
- **Migrating from another logger:** use [ADIF Import/Export](./adif-import-export/) before hand-entering old contacts.
- **Self-hoster:** follow [Self-Hosting](./self-hosting/) for Docker setup, backups, and update flow.
- **Portable activator:** read [POTA and SOTA Guide](./pota-sota/) for activation logging and references.
- **API or integration work:** jump to [API Reference](./api-reference/) and the desktop-client docs for WSJT-X/N1MM workflows.
